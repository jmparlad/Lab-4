Linear Regression Analysis

library(Lab4Package)

Overview

This vignette illustrates how the package Lab4Package works and it explains its different functionalities for handling linear regression models and making a simple regression analysis. The package is built using RC Object Oriented Programming, so that after initializing a class the user can have direct access to the different regression parameters and use the different functionalities of the class in an easy way.

Initializing the class

The first step for using the package is assigning the class. To do this, a data set and a formula or relation between the model variables should be introduced as input. This can be done as in the following example, where the well known data set iris is used:

data(iris)
linreg_ex <- linreg$new(Petal.Length ~ Species, data = iris)

Once the class is initialized, the regression parameters and the class methods can be called with simple commands.
